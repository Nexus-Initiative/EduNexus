  # EduNexus Product Roadmap

## Vision

EduNexus aims to become the operating system for African schools, centralizing academics, administration, communication, library management, and student development in one platform.

## Phase 1: MVP (June–August 2026)

Goal: Build a working school management platform and onboard the first pilot school.

### Student Information System

* Student profiles
* Parent profiles
* Teacher profiles
* Class management
* Subject management

### Academic Module

* Grade entry
* Grade viewing
* Report cards
* Transcripts

### Attendance Module

* Daily attendance
* Attendance reports
* Student attendance history

### Library Module

* Book catalog
* Book copies tracking
* Borrow records
* Return records
* Overdue tracking

### Parent Portal

* View grades
* View attendance
* View announcements

Success Metric:

* One pilot school onboarded
* At least 20 active users
